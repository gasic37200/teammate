## 📖 Project Overview
팀 프로젝트 모집 시 발생하는 정보의 불균형을 해결하기 위해, AI가 지원자의 포트폴리오와 경력을 분석하여 **객관적인 역량 등급**을 산출하고 최적의 팀원을 매칭하는 서비스입니다.

## 🛠 My Core Contributions (Team Lead)
- **[AI Engine]** `OpenAI` 및 `FactChat API`를 활용한 직군별(개발/기획/디자인) 역량 등급 산출 로직 설계
- **[Architecture]** `FastAPI` 기반 AI 전담 서버 구축 및 메인 서버 간 비동기 데이터 통신 인터페이스 설계
- **[Algorithm]** GitHub 커밋 데이터 및 PDF 포트폴리오 텍스트 파싱을 통한 실무 역량 지표(1~5단계) 자동화
- **[PM/QA]** 서비스 비즈니스 모델(헤드헌팅 방식) 기획 및 전체 매칭 로직 정합성 검증

## 🧠 AI Evaluation Logic (핵심 역량)
> **단순 키워드 매칭을 넘어선 정교한 역량 분석**
1. **Data Parsing:** 지원자의 GitHub 링크 및 PDF 포트폴리오 데이터 수집
2. **AI Analysis:** 직군별 필수 역량 지표(기술 스택, 프로젝트 경험, 문제 해결 능력)를 기준으로 AI 분석 수행
3. **Grading:** 수집된 정량/정성 데이터를 종합하여 AI 등급(Grade) 산출 및 팀별 적합도 리포트 생성

## 🚀 Key Experience
- **이종 프레임워크 간 협업:** Flutter(Frontend)와 FastAPI(AI Server) 간의 효율적인 API 프로토콜을 정의하며 시스템 설계 역량 강화
- **도메인 특화 AI:** 프롬프트 엔지니어링을 통해 AI가 전문적인 직군별 역량을 오차 없이 평가하도록 로직을 최적화한 경험
